Analytics Bağlantısı (OAuth)

Gerçek izlenme süresi ve trafik kaynağı verileri için Google OAuth Client ID gerekir. Bu, sayfanın http/https bir adreste (file:// değil) barındırılmasını gerektirir — Google'ın güvenlik kuralı. Google Cloud Console → Credentials → Create Credentials → OAuth client ID → "Web application" seçip, bu sayfayı barındırdığın adresi hem "Authorized JavaScript origins" hem de yönlendirme olarak eklemen yeterli.
